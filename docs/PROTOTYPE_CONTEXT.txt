PROJECT OVERVIEW
================================================================================

Skill-Bridge is an AI-powered career development platform that helps users 
bridge the gap between their current skills and their dream job.

TECH STACK:
- Frontend: Next.js 14, React, TypeScript, TailwindCSS
- Backend: Node.js, Express, MongoDB
- AI: OpenRouter API (GPT-3.5 Turbo)
- Styling: Dark mode, glassmorphism, neon accents

DESIGN PHILOSOPHY:
- Dark mode with deep purple/indigo gradients
- Glassmorphic cards with frosted glass effect
- Neon accent colors (cyan, purple, indigo)
- Premium SaaS aesthetic with smooth animations
- Mobile-first responsive design

COLOR PALETTE:
- Background: Dark gradient (gray-950 → indigo-950 → gray-950)
- Primary: Indigo (indigo-400, indigo-500, indigo-600)
- Secondary: Purple (purple-400, purple-500)
- Accent: Cyan (cyan-400), Pink (pink-400)
- Success: Green (green-400)
- Warning: Yellow (yellow-400)
- Error: Red (red-400)

================================================================================
TECHNICAL IMPLEMENTATION NOTES
================================================================================

FRONTEND STACK:
- Framework: Next.js 14 (App Router)
- Language: TypeScript
- Styling: TailwindCSS
- UI Components: Shadcn/ui
- Animations: Framer Motion
- Icons: Lucide React

BACKEND STACK:
- Runtime: Node.js
- Framework: Express
- Database: MongoDB
- AI: OpenRouter API (GPT-3.5 Turbo)
- File Processing: pdf-parse, mammoth

API ENDPOINTS:
1. POST /api/resume/upload - Upload and extract skills
2. POST /api/analyze/gap - Analyze skill gaps
3. POST /api/roadmap/generate - Generate 6-week roadmap

DATA FLOW:
1. Resume → Text Extraction → AI Analysis → Skills JSON
2. Skills + Job Description → AI Analysis → Gaps + Score
3. Gaps + Hours → AI Generation → 6-Week Roadmap

PERFORMANCE:
- Resume analysis: ~5-10 seconds
- Gap analysis: ~10-15 seconds
- Roadmap generation: ~30-40 seconds
- Total: ~1-2 minutes for complete flow

RESPONSIVE BREAKPOINTS:
- Mobile: < 640px (1 column, stacked)
- Tablet: 640px - 1024px (2 columns)
- Desktop: > 1024px (full layout)
